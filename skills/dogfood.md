# Dogfood Skill (Exploratory Testing)
This skill guides SimpleClaw through systematic exploratory testing of a web application.

## Goal
Discover UI bugs, broken links, and UX issues by navigating like a real user.

## Workflow
1. **Navigate** to the application homepage.
2. **Snapshot** to see available actions.
3. **Iterate**: Click buttons, fill forms, and verify transitions.
4. **Report**: Summarize findings with screenshots if possible.

## Example
"Test the login flow of myapp.vercel.app"
- Navigate to myapp.vercel.app
- Use browser to find login button
- Type test credentials
- Verify successful login
